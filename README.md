# Sandbox

A general-purpose workspace for building, testing, and experimenting with projects. Nothing here is precious — it's where ideas get prototyped, tools get tested, and throwaway code goes to live its best temporary life.

## What's in here

Various projects in different stages of completion (or abandonment). Each subdirectory is its own thing — some are full apps, some are one-off scripts, some are just seeing what happens if you do *that*.

## Rules of the sandbox

- Break things freely
- Don't commit secrets
- Clean up when you remember to
- If it works, it might graduate to its own repo
